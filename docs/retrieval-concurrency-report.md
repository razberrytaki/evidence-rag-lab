## 메모

- PostgreSQL 검색 동시성만 분리하기 위해 시간 측정 전에 임베딩을 의도적으로 미리 계산한다.
- 각 동시성 수준은 같은 검색 평가 사례를 실행한다.
- Warm/cold 캐시 분리는 하지 않는다. 동시성 1/4에서 DB 검색 구간만 비교한다.
